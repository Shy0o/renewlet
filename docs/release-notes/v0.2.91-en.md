# Renewlet v0.2.91

Renewlet v0.2.91 adds DingTalk bot notifications ([#51](https://github.com/zhiyingzzhou/renewlet/issues/51), thanks [@i7i777](https://github.com/i7i777)). You can send renewal reminders to a DingTalk group and match the bot's keyword or signing security settings.

## Read Before Upgrading

Cloudflare one-click deploy users should still upgrade from the generated repository by running `Sync Renewlet Upstream`. Do not click the one-click deploy button again. If an older generated repository does not have this entry yet, add it once from the [Cloudflare deploy documentation](https://github.com/zhiyingzzhou/renewlet/blob/main/docs/cloudflare-workers-deploy.md#existing-one-click-deploy-users).

---

## New Features

- Settings now includes a DingTalk bot notification channel for sending renewal reminders through a bot Webhook URL.
- DingTalk notifications support Markdown or text messages, with test sending from Settings.
- Title and body templates can include reminder title, content, send time, and item count variables.
- Keyword checks and a signing secret are supported for common DingTalk group bot security settings.
