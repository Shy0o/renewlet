# Renewlet v0.2.91

Renewlet v0.2.91 新增钉钉机器人通知（[#51](https://github.com/zhiyingzzhou/renewlet/issues/51)，感谢 [@i7i777](https://github.com/i7i777)）。可以把续费提醒发送到钉钉群，并按群机器人的安全设置配置关键词或加签。

[English ->](./v0.2.91-en.md)

---

## 升级前请先读

Cloudflare 一键部署用户仍然从生成仓库运行 `Sync Renewlet Upstream` 升级，不要重新点一键部署按钮。旧生成仓库如果还没有这个入口，按 [Cloudflare 部署文档](https://github.com/zhiyingzzhou/renewlet/blob/main/docs/cloudflare-workers-deploy.zh-CN.md#旧一键部署用户) 一次性补上。

---

## 新功能

- 设置页新增钉钉机器人通知渠道，可填写机器人 Webhook URL 后发送续费提醒。
- 钉钉通知支持 Markdown 或文本消息，并可直接在设置页测试发送。
- 支持自定义标题和正文模板，可插入提醒标题、正文、发送时间和提醒项目数量等变量。
- 支持钉钉关键词校验和加签密钥，适配群机器人的常见安全设置。
